# Stibbard's blog

Built with [MultiTerm Astro](https://github.com/stelcodes/multiterm-astro) by [Stel Clementine](https://github.com/stelcodes).

## Getting started

Clone the repository and install dependencies

```shell
git clone https://github.com/mstibbard/mstibbard.github.io.git blog && cd $_
bun install
```

### Local development

```shell
bun run dev
```

Note: Development will include all published content **and** draft content.

### Local build and preview

```shell
bun run build && bun run preview
```
